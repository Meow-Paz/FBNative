# FBNative
Rewrite Fastbuilder in C++.  
Supported Platforms: Linux,macOS(,Windows,iOS:Hardly but possible)
# Usage
Usage is just like FastBuilder,use CCmd(Chat Command) to control,with a simple "-" in the head.  
like `-round -r 6`.  
## Command List
```
-round (-r)
-circle (-r)
-torus (-r -w)
```
